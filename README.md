# TypeScript 함수형 프로그래밍 연습 프로젝트

## 깃 커밋 컨벤션
### 형식
```
<type>(<scope>): <subject>

- <description>
...
```

### 타입
- feat: 기능 구현
- fix: 버그/오류 수정
- refactor: 코드 개선, 리팩토링
- perf: 성능 개선
- chore: 환경 설정 변경
- docs: 문서 작성
- style: 코드 스타일 변경
- design: CSS 등 디자인 변경

### 예시
```
feat(auth): JWT 토큰을 활용한 로그인/로그아웃 구현

- JWT 토큰 활용하여 액세스 토큰, 리프레쉬 토큰 생성
- 리프레쉬 토큰으로 액세스 토큰 재발급 구현
- API 엔드 포인트마다 사용자 권한 검사 구현
```
